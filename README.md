# 1.4.1.Projectile_trajectories
# Šāviņa trajektorijas aprēķins 

Šī programma aprēķina šāviņa trajektoriju, izmantojot ievadītos mainīgos parametrus un dotajos nosacījumos noteiktos aprēķinus. <br>
Trajektorija tiek rēķināta pa soļiem līdz brīdim, kad šāviņš sasniedz mērķi vai nokrīt uz zemes. Programma izvada informāciju par trajektorijas punktiem aprēķina gaitā.<br>
Aprēķins tiek pārtraukts tikai tad, kad pēdējais aprēķinātais punkts pieder mērķim vai zemei, bet priekšpēdējais punkts atrodas gaisā. <br>
